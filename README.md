# official-main-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面作为独立的文档或工具页面存在，不隶属于任何特定域名或网站。

## 项目简介

这是一个通用型 HTML 页面归档仓库。仓库内的每个页面均为独立完整的 HTML 文件，可直接在浏览器中打开使用。页面内容涵盖笔记、工具、参考信息等多种类型，旨在提供轻量、自包含的网页资源。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立的页面，文件名即页面标识。
- `assets/`：存放页面间共享的静态资源，保持资源复用与整洁。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，部分页面可能引用 `assets/` 下的公共资源。
- 页面内容由仓库维护者不定期添加或更新。
- 页面不依赖外部服务或特定域名，可离线或本地浏览。

## 维护说明

- 本仓库由维护者根据需求进行更新。
- 新增页面请放入 `pages/` 目录，并更新 `index.html` 中的导航链接。
- 公共资源如有变更，请确保不影响已有页面的正常显示。
- 欢迎提交 Issue 或 Pull Request 讨论页面内容或结构改进。
